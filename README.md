# crud-golang-rabbitmq-mongo

## Stack

- [Golang](https://go.dev/)
- [RabbitMQ](https://www.rabbitmq.com/)
- [MongoDB](https://www.mongodb.com/)


To run and stop the applications:

By Makefile: 
```
make start

make stop
```
